ARRAYS

1.The isArray() method determines whether an object is an array.This function returns true if the object is an array, and false if not. e.g. function myFunction() { var fruits = ["Banana", "Orange", "Apple", "Mango"]; var x = document.getElementById("demo"); x.innerHTML = Array.isArray(fruits); } o/p: true
2.The join() method returns the array as a string.The elements will be separated by a specified separator. The default separator is comma (,). e.g. var fruits = ["Banana", "Orange", "Apple", "Mango"]; var energy = fruits.join(); o/p: Banana,Orange,Apple,Mango

STRINGS

1.The concat() method is used to join two or more strings.This method does not change the existing strings, but returns a new string containing the text of the joined strings. e.g.var str1 = "Hello "; var str2 = "world!"; var res = str1.concat(str2); o/p:Hello world!
2.The toUpperCase() method converts a string to uppercase letters. e.g.var str = "Hello World!"; var res = str.toUpperCase(); o/p:HELLO WORLD!
